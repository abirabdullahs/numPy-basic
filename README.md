# NumPy Learning Series

This repository contains **9 Python files** designed to help you learn **NumPy** step by step. Each file focuses on a specific concept, and every `print()` statement is followed by a `# output: ...` comment showing the actual output. This allows you to understand the code even without running it.

---

# File List

| File Name | Topics Covered |
|------------|----------------|
| `1_intro_to_numpy.py` | What is NumPy, Importing NumPy, List vs Array, 1D/2D/3D Arrays |
| `2_ndarray_attributes.py` | `shape`, `ndim`, `size`, `dtype`, `itemsize`, `nbytes` |
| `3_data_types.py` | Integer, Float, Boolean, String Arrays, `dtype=`, `astype()`, Type Promotion |
| `4_creating_arrays.py` | Creating Arrays from Lists/Tuples, Nested Lists, Creating New Arrays from Existing Arrays |
| `5_random_arrays.py` | `rand()`, `randint()`, `randn()`, `choice()`, `seed()` |
| `6_range_reshape.py` | `arange()`, `linspace()`, `reshape()`, `reshape(-1)` |
| `7_indexing_slicing.py` | Positive/Negative Indexing, 2D Indexing, All Types of Slicing |
| `8_copy_view_advanced_indexing.py` | `copy()`, `view()`, Boolean Indexing, Fancy Indexing, Looping Through Arrays |
| `9_common_functions.py` | `sum()`, `mean()`, `max()`, `min()`, `argmax()`, `argmin()`, `sort()`, `unique()`, `axis` |

# How to Run

## 1. Install NumPy

```bash
pip install numpy
```

## 2. Run a Python File

```bash
python 1_intro_to_numpy.py
```

Replace the filename with any of the other lesson files to explore different topics.
