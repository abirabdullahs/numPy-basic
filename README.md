# NumPy Learning Series 📘

এই ফোল্ডারে NumPy শেখার জন্য ৯টা Python ফাইল আছে। প্রতিটা ফাইল একটা নির্দিষ্ট concept কভার করে এবং প্রতিটা `print()` লাইনের নিচে `# output: ...` আকারে আসল output দেওয়া আছে — তাই ফাইল রান না করেও output দেখা যাবে।

## ফাইল লিস্ট

| ফাইল নাম | Topic |
|---|---|
| `1_intro_to_numpy.py` | NumPy কী, import, List vs Array, 1D/2D/3D Array |
| `2_ndarray_attributes.py` | shape, ndim, size, dtype, itemsize, nbytes |
| `3_data_types.py` | Int/Float/Bool/String Array, dtype=, astype(), Type Promotion |
| `4_creating_arrays.py` | List/Tuple থেকে Array, Nested List, Existing Array থেকে নতুন Array |
| `5_random_arrays.py` | rand(), randint(), randn(), choice(), seed() |
| `6_range_reshape.py` | arange(), linspace(), reshape(), reshape(-1) |
| `7_indexing_slicing.py` | Positive/Negative Indexing, 2D Indexing, সব ধরনের Slicing |
| `8_copy_view_advanced_indexing.py` | copy(), view(), Boolean Indexing, Fancy Indexing, Loop |
| `9_common_functions.py` | sum, mean, max, min, argmax, argmin, sort, unique, axis |

## কীভাবে রান করবেন

```bash
pip install numpy
python 1_intro_to_numpy.py
```

## প্রয়োজনীয়তা

- Python 3.x
- NumPy (`pip install numpy`)
